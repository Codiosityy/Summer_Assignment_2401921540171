### 📅 Week 1: Foundations of Problem Solving

#### 🌅 Day 1: Arrays & Hashing

A great start to the journey! Solved **3 classic array problems** that introduce fundamental techniques like hashing, two-pointers, and single-pass optimization.

| # | Problem | Difficulty | Topic | Time | Space | Solution | LeetCode |
|:-:|---------|:----------:|-------|:----:|:-----:|----------|:--------:|
| **1** | [Two Sum](https://leetcode.com/problems/two-sum/description/) | 🟢 Easy | Hash Map | `O(n)` | `O(n)` | [📄 C++](./Week_1/Day_1/1.cpp) | [🔗 Link](https://leetcode.com/problems/two-sum/) |
| **26** | [Remove Duplicates from Sorted Array](https://leetcode.com/problems/remove-duplicates-from-sorted-array/description/) | 🟢 Easy | Two Pointers | `O(n)` | `O(1)` | [📄 C++](./Week_1/Day_1/2.cpp) | [🔗 Link](https://leetcode.com/problems/remove-duplicates-from-sorted-array/) |
| **121** | [Best Time to Buy and Sell Stock](https://leetcode.com/problems/best-time-to-buy-and-sell-stock/description/) | 🟢 Easy | Sliding Window | `O(n)` | `O(1)` | [📄 C++](./Week_1/Day_1/3.cpp) | [🔗 Link](https://leetcode.com/problems/best-time-to-buy-and-sell-stock/) |

> 💡 **Highlights**: All three problems are LeetCode Easy-rated, focusing on building strong fundamentals in array manipulation and optimization. Each demonstrates a different core technique.

<details>
<summary><b>📌 Click to see problem summaries</b></summary>

<br/>

**Problem 1: Two Sum** — Given an array of integers and a target, return indices of two numbers that add up to the target.
> **Approach**: Hash map to store seen values and their indices. Single pass, look up complement in O(1).

**Problem 26: Remove Duplicates from Sorted Array** — Remove duplicates in-place from a sorted array, return the new length.
> **Approach**: Two pointers — `read` scans the array, `write` overwrites duplicates. O(1) extra space.

**Problem 121: Best Time to Buy and Sell Stock** — Find the maximum profit from one buy-sell transaction.
> **Approach**: Track the minimum price seen so far, compute max profit in a single pass.

</details>

| Type | Pattern | Example |

| LeetCode Solution | `<problem-number>.cpp` | `1.cpp` → LeetCode #1 |

1. **Pick the right folder** → `Week_X/Day_Y/`
2. **Name the file** → Use the LeetCode problem number: `<number>.cpp`
3. **Write clean code** → Use the standard `Solution` class, add comments for approach & complexity
